LM Studio supports multiple inference runtimes for running large language models locally. Each runtime has different strengths depending on your hardware and use case.

## Available Runtimes

- [**llama.cpp**](runtimes/llama-cpp) – A widely-used, cross-platform runtime for running GGUF models efficiently on CPUs and GPUs.
- [**MLX**](runtimes/mlx) – An Apple Silicon-optimized runtime for running models with high performance on Mac.
