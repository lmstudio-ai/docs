You can serve local LLMs from LM Studio's Developer tab, either on localhost or on the network.

LM Studio's APIs can be used through an [OpenAI compatibility mode](/docs/api/endpoints/openai), ehanced [REST API](/docs/api/endpoints/rest), or through a client library like [lmstudio.js](/docs/api/sdk).

#### API options

- [OpenAI Compatibility endpoints](/docs/api/endpoints/openai)
- [LM Studio REST API](/docs/api/endpoints/rest) (new, in beta)
- [TypeScript SDK](/docs/api/sdk) - `lmstudio.js`
