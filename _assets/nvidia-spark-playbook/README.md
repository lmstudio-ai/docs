# Running the scripts

## Javascript

Pre-reqs: User has installed `npm` and `node`

```bash
cd js
npm install @lmstudio/sdk
node run.js
```

## Python

Pre-reqs: User has installed `uv`

```bash
=
uv run --script run.py
```

## Bash

Pre-reqs: User has installed `jq` and `curl`

```bash
bash run.sh
```
